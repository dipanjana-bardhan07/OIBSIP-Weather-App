# Task: Command-Line Weather Application

A Python-based application that fetches and displays live weather data for any requested city. This project demonstrates how to connect to external REST APIs, handle JSON responses, and display clean data to the end user.

## Features
* **Real-time Data:** Fetches current temperature, humidity, and weather descriptions.
* **Global Search:** Allows users to input any city name globally to check the conditions.
* **API Integration:** Utilizes the HTTP requests protocol to fetch live weather telemetry.
